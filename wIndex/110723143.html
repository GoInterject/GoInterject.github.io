Function Summary

Acts as a reference point for ReportMerge . For examples and discussion, view the tutorial.

Function Arguments

Parameter Name
Description
Default
Optional

TabName

Enter the name of the tab to be copied. Supports a single variable (example "Sheet1", "Sheet*", or just "*")

NO

Placement

Enter "before" or "after" to indicate the placement of the sheet based on the anchor sheet


NO
Anchor Enter the name of the sheet to be used as the placement anchor. Also use jTabName() to indicate the current sheet, or use jTabName(Sheet2!a2) to reference another sheet.
NO
